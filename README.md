## 📌 About

Cashlytics is a comprehensive personal finance management platform built entirely on the frontend — no servers, no cloud, no privacy concerns. All your financial data lives on your device using IndexedDB.

What started as a simple expense tracker evolved into a full financial portfolio manager covering accounts, investments, EMIs, and tax-saving schemes.

---

## ✨ Features

- **Multi-Account Management** — Track savings, current accounts, credit cards, loans, and cash together
- **Investment Tracking** — Monitor stocks, mutual funds, and SIPs with profit/loss calculations
- **EMI & Loan Manager** — Automated EMI calculations with payment schedules and projections
- **Tax-Saving Schemes** — Manage PPF, FD, RD, NSC with maturity date tracking
- **Visual Analytics** — Charts and graphs for spending patterns, savings rate, and net worth
- **100% Offline & Private** — Data stored locally via IndexedDB, zero cloud dependency

## 🚀 Getting Started

No installation needed. Just open in a browser.

**Option 1 — Direct open**
Download the project → open index.html in any browser



**Option 2 — Clone and run**
```bash
git clone https://github.com/raghavqk/cashlytics.git
cd cashlytics
# Open index.html in your browser
```

**Option 3 — Live Demo**

Visit the live version on GitHub Pages: [raghavqk.github.io/cashlytics](https://raghavqk.github.io/cashlytics)
